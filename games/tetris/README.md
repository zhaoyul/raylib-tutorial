# 俄罗斯方块 / Tetris

## 游戏简介 / Game Description

经典俄罗斯方块游戏。旋转和移动方块填满整行。

Classic Tetris game. Rotate and move blocks to complete rows.

## 游戏特性 / Features

- 7种方块形状
- 方块旋转
- 消行系统
- 分数和等级
- 预览下一个方块

## 操作说明 / Controls

- **左右方向键 / A-D**: 移动方块
- **上方向键 / W**: 旋转方块
- **下方向键 / S**: 快速下落
- **空格键**: 直接落下
- **ESC**: 返回菜单
