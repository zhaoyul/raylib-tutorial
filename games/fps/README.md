# 第一人称射击游戏 / First-Person Shooter

## 游戏简介 / Game Description

简单的第一人称射击游戏。使用3D相机和基本射击机制。

Simple first-person shooter game with 3D camera and basic shooting mechanics.

## 游戏特性 / Features

- 第一人称视角
- 3D环境漫游
- 射击目标
- 简单的得分系统

## 操作说明 / Controls

- **WASD**: 移动
- **鼠标**: 视角控制
- **鼠标左键**: 射击
- **ESC**: 返回菜单
