# Snake 游戏版本目录

本目录包含 Snake 游戏的各个渐进式版本。

## 目录结构

```
phases/
├── README.md           # 本文件
├── v0-base/           # 基础版本
├── v1-items/          # 道具系统 ✅
├── v2-fx/             # 粒子特效 (TODO)
├── v3-audio/          # 音效存储 (TODO)
└── v4-multi/          # 多人模式 (TODO)
```

## 版本状态

| 版本     | 状态      | 描述                                            |
|----------|-----------|-------------------------------------------------|
| v0-base  | ✅ 完成   | 经典贪吃蛇，位于上级目录 `games/snake/main.cpp` |
| v1-items | ✅ 完成   | 多种食物类型、障碍物系统、继承多态              |
| v2-fx    | ✅ 完成   | 粒子效果、屏幕震动、缓动函数                    |
| v3-audio | 📝 待实现 | 音效、高分榜、设置                              |
| v4-multi | 📝 待实现 | 双人模式、关卡编辑器                            |

## 如何开始

1. **学习现有代码：** 查看 `../main.cpp` (v0-base)
2. **阅读路线图：** 查看 `../ROADMAP.md`
3. **开始扩展：** 创建新目录，如 `v1-items/`，复制基础代码开始修改

## 贡献

如果你想实现某个版本：
1. Fork 本仓库
2. 在对应版本目录创建代码
3. 更新版本状态
4. 提交 PR
