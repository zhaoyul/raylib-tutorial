# 打砖块游戏 / Brick Breaker

## 游戏简介 / Game Description

经典的打砖块游戏。控制挡板，让球反弹并击碎所有砖块。

Classic brick breaker game. Control the paddle to bounce the ball and break all bricks.

## 游戏特性 / Features

- 挡板控制
- 球的物理反弹
- 多层砖块
- 分数系统
- 生命系统

## 操作说明 / Controls

- **左右方向键 / A-D**: 移动挡板
- **空格键**: 发射球
- **P**: 暂停游戏
- **ESC**: 返回菜单

## 学习要点 / Learning Points

- 碰撞检测应用
- 物理模拟
- 游戏状态管理
- UI 绘制
